### Tipos de Variáveis

<details>
<summary>Tipos de Variáveis:</summary>

- char = %c;
- int = %d;
- long int = %ld;
- float = %f, %2f;
- double = %if;

</details>

### Scaneando
 <summary>Como escanear:<\summary>
 - scanf("%d" , variavel),


```ruby
exemplo
#include <stdio.h>
void main() {
    int x = 0;

    printf("numero:");
    scanf("%d", &x);
    int variavel = x+ 9;
    printf("%d", variavel);
}
```
